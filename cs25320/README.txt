ASTER-RISK WEBSITE AND GAME - CS25320 ASSIGNMENT
------------------------------------------------------------------------------------------------
The following zip contains the .html files as well as the css file and the images and sounds 
that are needed in order to run the game in the first place.

To start the game, please click the index.html file and click the "Play Game" button that will
appear on the site promptly.


.HTML FILES
------------------------------------------------------------------------------------------------
Index - The main index of the site, contains 3 buttons that will take you to the other files
shown below.

About - The about page gives credit for the images, sounds and snippets of code that are used 
within the game. 

Help - The help page briefly describes the objective of the game and the controls.
       Mouse movement/left-right arrow keys to move, Mouse click/space to shoot.

PlayGame - The main game. Click the Play Game button to initiate the game.
